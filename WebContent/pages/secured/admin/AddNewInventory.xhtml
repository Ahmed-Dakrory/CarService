Notes:

Selecting freight type
Please note that for automobiles, boats, motorbikes, scooters, etc. “Vehicle” Type should be selected.

Checking VIN/PO#
Please note it is required to check VIN/PO#. After checking VIN/PO# you can add further information for your freight.
If your VIN is found in our external VIN database, vehicle information is added automatically. Otherwise, please fill in the fields manually.
For a vehicle, it is required to indicate if it is Self-propelled or Immobile.

Description
For dry cargo, it is required to input description.

Choosing destination
Please select destination in three steps: 1) choose country; 2) choose state (if any); 3) choose port or inland destination.
If you have not found your destination in this list, please enter it in the field “Notes” at the bottom of the page.

Consignee Info
You can select consignee for a new freight from the list of your consignees. For adding a new consignee, please go to Consignee List Tab.

Picture Request
Please tick the box if you require pictures of your freight.
